# 4P3X Refractable Base OS™ — Safety & Disclaimer Module

**4P3X Refractable Base OS™ — Participant PWA + Admin Monitoring Dashboard**
Powered by 4P3X Intelligent AI — Created by Kyzel Kreates

---

## Purpose

4P3X Refractable Base OS™ supports activity tracking, evidence organisation, and participant progress monitoring workflows.

This document defines the safety rules, scope boundaries, and disclaimers for the embedded 4P3X Intelligent AI assistants.

---

## AI Disclaimer

> "4P3X Intelligent AI provides guidance, activity support, and dashboard help only. It does not replace a qualified professional, manager, clinician, legal adviser, mentor, or compliance officer. All important decisions should be reviewed by an appropriate human support person."

---

## What the AI Assistants Can Do

- Help admins navigate the monitoring dashboard
- Help admins understand participant activity metrics and progress patterns
- Help participants navigate the Participant PWA
- Help participants log activity, complete check-ins, and submit evidence
- Provide neutral, non-clinical support guidance to participants
- Suggest next steps based on activity data
- Explain progress summaries and milestone status

---

## What the AI Assistants Cannot Do

- Make compliance, legal, or regulatory decisions
- Provide clinical, medical, or therapeutic advice
- Replace a qualified professional, clinician, or legal adviser
- Guarantee any outcome for any participant
- Make decisions about benefit, entitlement, or legal compliance of any kind
- Access external systems, APIs, or third-party data
- Send messages, emails, or communications on behalf of users

---

## Scope Separation

Each of the 4 assistants has a strictly defined scope:

| Assistant | Scope |
|---|---|
| 4P3X Intelligent AI 1 — Admin Guide | Dashboard navigation and admin guidance only |
| 4P3X Intelligent AI 2 — Progress Insight | Participant data interpretation and insight only |
| 4P3X Intelligent AI 3 — Participant Guide | Participant PWA navigation and task guidance only |
| 4P3X Intelligent AI 4 — Support Assistant | Neutral, non-clinical participant support only |

Cross-scope requests are declined with a handoff to the appropriate assistant.

---

## Data Privacy

- All AI responses are generated from locally stored data only
- No data is sent to external AI APIs in local/rule-based mode
- In cloud AI mode, only anonymised activity summaries are used
- No personal identifiers are included in AI prompts in any mode

---

## Human Review Requirement

All AI outputs must be reviewed by an appropriate human support person before being used to make any decision about a participant's progress, support needs, or programme outcomes.
